8.23 Induction Example

8.23 记一道Induction题目

    The sequence of real numbers a1,a2,a3,…a_{1},a_{2},a_{3},\ldotsa1​,a2​,a3​,… is such that a1=1a_{1} = 1a1​=1 and

an+1=(an+1an)3a_{n+1}=\left(a_{n}+\frac{1}{a_{n}}\right)^{3} an+1​=(an​+an​1​)3

a. Prove by mathematical induction that ln⁡an≥3n−1ln⁡2\ln a_{n} \geq 3^{n - 1}\ln 2lnan​≥3n−1ln2 for all integers n≥2n \geq 2n≥2.

      [You may use the fact that ln⁡(x+1x)>ln⁡x\ln{\left( x + \frac{1}{x} \right) > \ln x}ln(x+x1​)>lnx for x>0x > 0x>0.]

Solution:

a.

              Let n=2n = 2n=2,

                                                      a2=(a1+1a1)3=13=8a_{2} = \left( a_{1} + \frac{1}{a_{1}} \right)^{3} = 1^{3} = 8a2​=(a1​+a1​1​)3=13=8,

                                                      ln⁡a2=ln⁡23=3ln⁡2≥3ln⁡2\ln{a_{2} = \ln{2^{3} = 3\ln 2 \geq 3\ln 2}}lna2​=ln23=3ln2≥3ln2

                                     So it’s true for n=2n = 2n=2.

              Assume it’s true for n=kn = kn=k:

                                                      ln⁡ak≥3k+1ln⁡2\ln{a_{k} \geq 3^{k + 1}\ln 2}lnak​≥3k+1ln2

                                     When n=k+1n = k + 1n=k+1:

                                                      ak+1=(ak+1ak)3a_{k + 1} = \left( a_{k} + \frac{1}{a_{k}} \right)^{3}ak+1​=(ak​+ak​1​)3

                                                      ln⁡ak+1=ln⁡(ak+1ak)3=3ln⁡(ak+1ak)>3ln⁡ak≥3∙3k−1ln⁡2 \ln{a_{k + 1} = \ln\left( a_{k} + \frac{1}{a_{k}} \right)^{3} = 3\ln{\left( a_{k} + \frac{1}{a_{k}} \right) > 3\ln{a_{k} \geq}3 \bullet 3^{k - 1}\ln 2\ }}lnak+1​=ln(ak​+ak​1​)3=3ln(ak​+ak​1​)>3lnak​≥3∙3k−1ln2 

                                                      ln⁡ak+1≥ 3∙3k+1∙ln⁡2\ln{a_{k + 1} \geq}\ 3 \bullet 3^{k + 1} \bullet \ln 2lnak+1​≥ 3∙3k+1∙ln2

                                                      ln⁡ak+1≥3k∙ln⁡2\ln a_{k + 1} \geq 3^{k} \bullet \ln 2lnak+1​≥3k∙ln2

                                     So it’s true for n=k+1n = k + 1n=k+1

              Hence by induction, it’s true for n≥2, n∈Z+n \geq 2,\ n \in Z^{+}n≥2, n∈Z+.
