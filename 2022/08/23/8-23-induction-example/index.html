8.23 Induction Example

8.23 记一道Induction题目

     The sequence of real numbers $ a_{1},a_{2},a_{3},\ldots$ is such that $a_{1} = 1 $ and

a. Prove by mathematical induction that $ \ln a_{n} \geq 3^{n - 1}\ln 2$ for all integers $n \geq 2 $.

       [You may use the fact that $ \ln{\left( x + \frac{1}{x} \right) > \ln x} $ for $x > 0$.]

Solution:

a.

               Let $ n = 2 $,

                                                       $ a_{2} = \left( a_{1} + \frac{1}{a_{1}} \right)^{3} = 1^{3} = 8 $,

                                                       $ \ln{a_{2} = \ln{2^{3} = 3\ln 2 \geq 3\ln 2}} $

                                      So it’s true for $n = 2$.

               Assume it’s true for $n = k$:

                                                       $ \ln{a_{k} \geq 3^{k + 1}\ln 2} $

                                      When $n = k + 1$:

                                                       $ a_{k + 1} = \left( a_{k} + \frac{1}{a_{k}} \right)^{3} $

                                                       $ \ln{a_{k + 1} = \ln\left( a_{k} + \frac{1}{a_{k}} \right)^{3} = 3\ln{\left( a_{k} + \frac{1}{a_{k}} \right) > 3\ln{a_{k} \geq}3 \bullet 3^{k - 1}\ln 2\ }} $

                                                       $ \ln{a_{k + 1} \geq}\ 3 \bullet 3^{k + 1} \bullet \ln 2 $

                                                       $ \ln a_{k + 1} \geq 3^{k} \bullet \ln 2 $

                                      So it’s true for $n = k + 1$

               Hence by induction, it’s true for $ n \geq 2,\ n \in Z^{+} $.
